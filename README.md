# ainft-js

A TypeScript SDK for the AINFT Server, a service by AI Network for NFT-related requests from
NFT bots, NFT frontends, and/or NFT devices.

## Requirements
- node >= 16

## ENVs
```
NODE_ENV=dev||prod
```

## Usage

### Develop
```bash
yarn install
yarn dev
```

### Production
```bash
yarn install
yarn build
```

## Tests
```bash
yarn test
```
